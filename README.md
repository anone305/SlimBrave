## Brave Browser Debloater (Batch Version)

SlimBrave Ultimate is a powerful Batch script designed for Windows users to streamline their Brave Browser experience. This enhanced fork expands the original tool to **36 comprehensive options** with a brand new **2-column interface** for better visibility and control.

### Key Enhancements:
* **Expanded Control:** Now features 36 togglable options across Telemetry, Privacy, and Performance.
* **New UI:** Optimized 2-column layout to see all settings at a glance.
* **Stability:** Fixed bugs related to parentheses in registry keys to prevent script crashes.
* **Undo System:** Toggle settings easily; unchecking a number reverts the change to default.

### Features:

- **Telemetry & Reporting** (Metrics, Safe Browsing, URL Collection, Feedback)
- **Privacy & Security** (Autofill, WebRTC, QUIC, Do Not Track, SafeSearch)
- **Brave Specific Features** (Rewards, Wallet, VPN, AI Chat, Shields, Tor, Sync)
- **Performance & Bloat** (Background Mode, Media Recs, Shopping, Translate, Spellcheck)
- **Advanced Restrictions** (Incognito Mode, Printing, Developer Tools, PDF Handling)
- **DNS Controls** (DNS Over HTTPS toggle)

# How to Run

1. Download the `SlimBrave.bat` file from this repository.
2. Right-click the file and select **Run as Administrator**.
3. Use the numbers to toggle features and press **A** to apply changes.
